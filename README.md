### linux 安装

cd ./vim/linux
./install

### macosx 安装

cd ./vim/macosx
./install

### windows 安装

将./vim/windows下面的文件，复制到gvim.exe对应的相关目录中
将./vim/vim下面的配置文件，复制到vim安装目录中
