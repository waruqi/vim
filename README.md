里面具体有哪些插件和具体使用方式，等以后有时间了再细说吧。

大家可以自己摸索下哈 :)

目前支持vim 7.4最新版，以及gvim/MacVim前端。。

### linux 安装

cd ./vim/linux
./install

### macosx 安装

cd ./vim/macosx
./install

### windows 安装

将./vim/windows下面的文件，复制到gvim.exe对应的相关目录中
将./vim/vim下面的配置文件，复制到vim安装目录中
